Zotero Prevent Duplicates Plugin

This plugin was developed by Cora Johnson-Roberson. It works for Zotero for Firefox only. Cora doesn't have time to maintain it, so I'm offering it a home (and a signed version) here, but I will not be developing it or (likely) even maintain it should it break in the future.

Note that Prevent Duplicates breaks import on Zotero 5.0 and I will not have time to update it for use with 5.0
